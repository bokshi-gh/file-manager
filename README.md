# File Manager (fm)

A simple and lightweight file manager for Linux written in C using GTK 4.

## Features

* Simple and intuitive interface
* Lightweight and fast
* Native GTK 4 user interface
* Built with modern C
* Follows Linux desktop conventions
* Open source

> **Note:** This project is currently under development.

## Dependencies

- GTK 4 (includes GLib and GIO)
- Meson
- Ninja
- GCC

## Building

Clone the repository:

```bash
git clone https://github.com/bokshi-gh/file-manager.git
cd file-manager
```

Configure the build directory:

```bash
meson setup build
```

Compile the project:

```bash
meson compile -C build
```

Run the application:

```bash
./build/fm
```
